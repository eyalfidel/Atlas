# 🌍 Atlas - האטלס הוויקיפדי

[![Website](https://img.shields.io/badge/Website-atlas.eyalfidel.com-blue?style=flat&logo=google-chrome)](https://atlas.eyalfidel.com)

**Atlas** היא פלטפורמה אינטראקטיבית ומתקדמת לחקר גאוגרפי, המשלבת ויזואליזציה של נתונים מרחביים (GIS) ומידע אנציקלופדי בזמן אמת. המערכת מאפשרת למשתמשים לחקור מדינות, ימים, ערים, אתרי טבע ומונומנטים היסטוריים על גבי גלובוס תלת-ממדי, תוך קבלת שכבות מידע סטטיסטיות ודמוגרפיות עשירות.

---

## ✨ תכונות מרכזיות (Features)

* **📊 חקר מבוסס נתונים (Data-Driven Exploration):** הצגת נתוני מאקרו-כלכלה ודמוגרפיה ברמת מדינה, כולל תמ"ג (GDP), מדד פיתוח אנושי (HDI), תוחלת חיים, גודל אוכלוסייה ושטח (כולל דירוג עולמי).
* **⚖️ מנוע השוואת מדינות:** כלי אנליטי המאפשר השוואה סטטיסטית של נתונים בין שתי מדינות ראש בראש בתצוגה טבלאית נוחה.
* **🗺️ מיפוי וסינון דינמי:** אלפי נקודות עניין (POIs) המחולקות לקטגוריות (ערי בירה, ערים מרכזיות, טבע, היסטוריה ומונומנטים) הניתנות לסינון בזמן אמת דרך מערכת מקרא חכמה.
* **🔍 חיפוש מתקדם (Advanced Search):** חיפוש מהיר המשלב פילטרים (חיפוש בקרב מדינות, ערים או אתרים) ואפשרות לחיפוש לוקאלי "בתוך המדינה הנוכחית בלבד".
* **🔗 שיתוף קישורים חכמים (Deep Linking):** יצירת כתובת URL ייחודית לכל מדינה או אתר ספציפי לשיתוף מהיר וממוקד.
* **📱 חוויית משתמש (UX) מותאמת מובייל:** ממשק רספונסיבי לחלוטין הכולל חלוניות מידע נגררות המותאמות למסכי מגע.

הפרויקט בנוי בארכיטקטורה המפרידה בין לוגיקת צד-הלקוח לבין שכבות המידע הגיאוגרפיות (GeoJSON) והסטטיסטיות.

| תפקיד / תיאור הקובץ | שם הקובץ / נתיב |
| :--- | :--- |
| קובץ הליבה של האפליקציה. מכיל את לוגיקת ה-Mapbox, ניהול אירועים (JS) ועיצוב הממשק (CSS). | `index.html` |
| תיקייה המרכזת את כלל מסדי הנתונים הגיאוגרפיים והסטטיסטיים של המערכת. | `json/` |
| נתוני דמוגרפיה, כלכלה (GDP, HDI) ומדדים אסטרטגיים ברמת מדינה. | `json/stats.json` |
| מאגר תקצירים מאנציקלופדיית ויקיפדיה שנשאבו מראש עבור מדינות, טריטוריות וימים. | `json/wiki_summaries.json` |
| שכבת נקודות מרחבית (Centroids) המשמשת כעוגן גיאוגרפי למדינות העולם. | `json/countries_points.geojson`|
| קואורדינטות ונקודות ציון של ימים ואוקיינוסים ברחבי הגלובוס. | `json/seas_points.geojson` |

| שכבת אתרים מיוחדים המחולקת לתתי-קטגוריות: טבע, היסטוריה ומונומנטים תרבותיים. | `json/places.geojson` |

## 🛠️ טכנולוגיות וארכיטקטורה (Tech Stack)

הפרויקט נבנה בארכיטקטורת צד-לקוח (Frontend) מתקדמת ללא תלות בשרת צד-אחורי מסורתי:

* **HTML5, CSS3, Vanilla JavaScript (ES6+)**
* **Mapbox GL JS:** מנוע רינדור מבוסס WebGL לתצוגת המפה והגלובוס בתלת-ממד.
* **Fetch API & Promises:** לטעינה אסינכרונית של קובצי הנתונים ושליפת תקצירים חיים מה-API של ויקיפדיה.

## 📚 מקורות מידע (Data Sources)

המערכת שואבת ומצליבה מידע ממספר מסדי נתונים:
1. **ויקיפדיה העברית (Wikipedia API):** משיכת תקצירים אנציקלופדיים בזמן אמת עבור ערים ואתרים נבחרים.
2. **ויקינתונים (Wikidata):** שאילתות SPARQL לאיסוף קואורדינטות, נתוני אוכלוסייה וסיווג גאוגרפי של אלפי יישובים ואתרים.
3. **Natural Earth:** נתוני בסיס ופוליגונים מפורטים להגדרת גבולות מדיניים וימים.
